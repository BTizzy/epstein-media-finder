# Utils Directory

This folder contains utility modules for the Epstein Media Finder tool:

- `doj_scraper.py`: DOJ website scraping utilities
- `media_processor.py`: Image hashing, metadata extraction, thumbnail generation
- `social_checker.py`: Free web scraping for social media presence
- `__init__.py`: Makes utils a Python package

These modules are imported by the pipeline scripts. See the main README for details.
